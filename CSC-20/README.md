# Programming Concepts and Methodology II (CSC 20)
This repository is dedicated to the CSC 20 course taught at Sacramento State by Dr. Gita Faroughi during the Spring 2021 Semester.

## Directory Focal Points
- Contains class assignments and examples practicing basic java fundamentals.
- Contains more complicated concepts than CSC-15, such as:
  - Objects.
  - Recursion.
  - Linked lists.
  - Queues.
