# Operating System Principles (CSC 139)
This repository is dedicated to the CSC 139 course taught at Sacramento State by Dr. Khaleq during the Fall 2025 Semester.

## Directory Focal Points
- Contains implementation examples/practice problems for:
  - Context switching
  - CPU scheduling
  - Process control and sys calls

## Personal Notes
- This was probably one of if not my favorite class I took at Sac State. Dr. Khaleq is a wonderful professor and made the concepts of this difficult course very digestable and manageable. I aspire to become a systems engineer or low-level/os engineer someday because of this class.
