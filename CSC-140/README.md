# Advanced Algorithm Analysis (CSC 140)
This repository is dedicated to the CSC 140 course taught at Sacramento State by Dr. Ghassan Shobaki during the Spring 2024 Semester and Spring 2026 Semester.

## Directory Focal Points
- Contains work for algorithmic problem solving such as Dijkistra implementation, Knapsack backtracking implementation, and some sorting algorithms.
- Work is separated into two directories for each time I took the class. Work may be unbalanced as some files were lost and can but have yet to be recovered.
- This class was also a part of my finals day from hell (Spring 2024 Semester), and I did not live to tell the tale like I can with 135. This class ate me up and spat me back out.
- All of his lectures for this class are available for free on his youtube account; just search his name and find his playlist on algorithms. While I think the class was still beneficial, you could totally learn all of the important concepts to a decent degree just from the videos (and maybe a little outside study/practice).
