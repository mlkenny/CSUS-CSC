# Object Oriented Computer Graphics Programming (CSC 133)
This repository is dedicated to the CSC 133 course taught at Sacramento State by Dr. Muyan-Oczelik during the Fall 2024 Semester.

## Directory Focal Points
- Contains CodeName1 files for a graphical game involving a spaceship floating through space and saving astronauts from aliens.
- The game follows the basic OOP principles and required a lot of UML diagram work as well as implemented very few design principles (singleton, observer, etc).
