# Introduction to Systems Programming in UNIX (CSC 60)
This repository is dedicated to the CSC 60 course taught at Sacramento State by Dr. Ruth Biel during the Spring 2023 Semester.

## Directory Focal Points
- Has examples of some basic C programs including:
  - Basic I/O.
  - File reading/writing.
  - Basic C logic.
  - Pointers.
  - Structs.
  - Makefiles.
  - Some sys calls.
