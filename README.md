# Sacramento State Computer Science Class Repository
This repository is intended to be a source of all of the class assignments, projects, labs, and some tests that I have created within my time at Sac State. This is not for distribution or cheating purposes as that would offend the Student Academic Dishonsty policy at the school, it is purely for collective/organizational and nostalgic purposes. Observe what you would like to observe, but all work stated here is mine, Michael Kenny, and mine alone and is not to be copy-pasted.

## Personal Notes
- Some classes are clearly missing, most noteably CSC-131, CSC-137, and CSC-159 (my third elective) and some assignments are missing as well, most noteably CSC-60 and CSC-138.
  - CSC-131 (Computer Software Engineering): This class was a project based class where we built a project as a team, and this project had a lot of weaknesses; such as exposed api key's.
  - CSC-137 (Computer Organization): This class was not a coding class but instead more of an electrical engineering course about logics gates and different hardware components and their interaction/importance within a system. No assignments to share as Dr. Singh was extremely straight forward.
  - CSC-159 (Operating System Pragmatics): This was the operating sys pragmatics class and had a much more complex code base, but I may add it at a later time. Dr. Kang Peng is my absolute GOAT.
  - CSC-60 Assignments: Most assignments were on paper; frankly, I don't remember much from the second half of this class because my professor broke her leg and was unable to come to class, so we had zoom classes and those sucked.
  - CSC-138 Assignments: This class, while it did have a handful of coding assignments, rarely ever had homework. Most "assignments" were just in class examples that I recreated. Actual assignments were on an online portal kind of like Pearson or Aleks, so not much to show here for that class.
