# Sacramento State Computer Science Class Repository
This repository is intended to be a source of all of the class assignments, projects, labs, and some tests that I have created within my time at Sac State. This is not for distribution or cheating purposes as that would offend the Student Academic Dishonsty policy at the school, it is purely for collective/organizational and nostalgic purposes. Observe what you would like to observe, but all work stated here is mine, Michael Kenny, and mine alone and is not to be copy-pasted.

## Personal Notes
- Some classes are clearly missing, most noteably CSC-131 and CSC-159 (my third elective), and some assignments are missing as well, most noteably CSC-60.
  - CSC-131: This class was a project based class where we built a project as a team, and this project had a lot of weaknesses; such as exposed api key's.
  - CSC-159: This was the operating sys pragmatics class and had a much more complex code base, but I may add it at a later time.
  - CSC-60 Assignments: Most assignments were on paper; frankly, I don't remember much from the second half of this class because my professor broke her leg and was unable to come to class, so we had zoom classes and those sucked.
